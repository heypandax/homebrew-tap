# heypandax/tap

```
brew install --cask heypandax/tap/cc-pocket
```

The Mac daemon for **CC Pocket** — drive Claude Code from your phone.
Prebuilt + notarized (no Xcode Command Line Tools, no Java). Apple Silicon.

Prereq: [Claude Code](https://claude.com/claude-code) installed + signed in.

> Homebrew may print a `tap not trusted` notice during install — it's a harmless
> transition warning and the install still works. To silence it (and stay working
> when Homebrew makes trust mandatory): `brew trust --cask heypandax/tap/cc-pocket`.
